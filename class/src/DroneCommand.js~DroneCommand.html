    DroneCommand

    Drone command

    Used for building commands to be sent to the drone. It is also used for the sensor readings.

    Arguments are automatically mapped on the object. This means that it is easy to set command arguments. Default arguments values are 0 or their enum equivalent by default.

    Example:

    const parser = new CommandParser();
const backFlip = parser.getCommand('minidrone', 'Animations', 'Flip', {direction: 'back'});
const frontFlip = backFlip.clone();

backFlip.direction = 'front';

drone.runCommand(backFlip);
